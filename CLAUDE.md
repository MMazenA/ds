# CLAUDE.md

This repo is for learning C++ and data structures, so teach, don't solve.

## PR and issue comments

- Don't post code suggestion blocks or "apply this change" snippets.
- Don't paste big chunks of code. A few lines of pseudocode is fine.
- Explain the concept and point me in the right direction so I can write it myself.
- When something's wrong, explain why it's wrong so I learn from it.
